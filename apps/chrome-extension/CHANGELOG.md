## [1.0.3] - 2026-07-02

### Current Branch: `build`

### Major Features
- 📥 Download Lark Docs as Markdown (with ZIP packaging for images/files)
- 📋 Copy Lark Docs as Markdown to clipboard
- 🌐 Download Lark Docs as HTML (with inline images)
- 👁️ View Lark Docs as Markdown preview
- 📊 Download Standalone Bitable as Markdown/HTML
- 🐛 Built-in Bug Report system (auto-generates GitHub Issues)

### Supported Block Types (20+)
- Headings (Level 1-9), Code blocks, Quotes, Lists (bullet/ordered/task)
- Tables, Images, Inline Math, Callouts, Synced References
- Iframes, Grids, Whiteboards, Diagrams, Bitable (embedded & standalone)
- And more...

### Inline Formatting
- Bold, Italic, Delete, Inline code, Links, Font color, Background color, Underline

### Technical Highlights
- Vue 3 + TypeScript + TailwindCSS architecture
- Monorepo with pnpm + Turborepo
- i18n support (English & Chinese)
- Comprehensive E2E testing with Playwright
- Automatic image compression and inline embedding
- Virtual scroll handling for long documents
- Error reporting with auto-generated GitHub Issues
