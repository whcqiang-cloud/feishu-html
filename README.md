Cloud Document Converter is a browser extension that supports downloading/copying Lark Doc as Markdown.

**English** · [简体中文](./README_ZH.md)

# Install

1. Go to the [Releases](https://github.com/whcqiang-cloud/feishu-html/releases) page
2. Download `cloud-document-html-{version}.zip` from the latest release
3. Open `chrome://extensions/` in Chrome, enable **Developer mode**
4. Unzip the file and drag the folder into the extensions page, or click **Load unpacked** and select the unzipped folder

# Features

- Download Lark Doc as Markdown

- Copy Lark Doc as Markdown

  > ❗Note: The URL of the image generated by this way only has two hours of accessibility. This means that after two hours, the image resource will no longer be accessible.

- Download Lark Doc as HTML

- Copy Lark Doc as HTML

- Download Standalone Bitable as Markdown

- Download Standalone Bitable as HTML

  > 💡Standalone Bitable export uses the browser's current login session to fetch data from Feishu's internal APIs. It works for tables loaded in the current browser tab.
